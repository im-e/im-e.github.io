Robot AI Programming

Robotics was a third year university module where the main assement piece of the module was to program a robot that has an advanced collision detection system, using LUA scripting inside CoppeliaSim(V-REP). I was tasked to create several states for the AI and create a logic-flow that goes between each state to give the AI a completed behaviour. The AI was then tested to be able to consistently traverse a room, without hitting and obstacles.

Features

  • Collision detection and avoidance
  • AI state machine - wander, avoid, follow/track states
  • Edge tracking/following

    • Mandlebrot Explorer

    Mandlebrot Explorer is an interactive viewer for the Mandlebrot set made inside Unity. It is styled with shaders to make the set fun to browse and look around in.
    I created it for fun, as I am really interested by complex dynamics and enjoy the art you can make with sets of complex numbers.

    Features

  • Stylistic shading and color cycling
  • Viewing controls with zooming and panning
  • A great way to experience the Mandlebrot set!
