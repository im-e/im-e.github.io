Soulflame

Responsive Platformer

Soulflame is a singleplayer 2D pixel platformer, inspired by Celeste and Super Meat Boy with fast and responsive movement.
It is independently developed by myself in Unity using C#. It was a development challenge to create fast paced platformer game within a 2 week timeframe, focusing on fluid but tight controls.
The player is put against the clock to kindle the soulflames of the world!

Features

  • Responsive Movement - Tight, fast and fluid movement that you cant get enough of
  • Advanced Techniques - Walljumping, Air-Dash, Variable jumping height
  • Overworld - Interactive menu and level select as an overworld
  • Progression System - Levels are unlocked progressively, based on level finishes and speed of finish
  • Replayable Levels - Players must improve their times to kindle the flame stronger and progress in the game
  • Records and Ranking System - High score leaderboards for times on each level
  • Saving and Loading System - Save states for records and progression
  • Cozy World - A lighthearted story set in a nostalgic retro style univerise
