Realistic Terrain

Advanced Shader Programming

Realistic Terrain was a university project for the Advanced Shader Programming module in my third year which required me to create a realistic piece of looking terrain created from a 50x50 plane using advanced shader programming techniques.

Features

  • Applied a height map of the swiss alps from Terrain.party, for realistic geometry.
  • Plane is illuminated with basic phong lighting
  • Plane is coloured based on the height of the vertex, to show where snow, rock and grass would lay.
  • Angles of the geometry also change the colouring, to show areas where snow would settle and grass would grow.
  • Distance dependent tesselation to increase and decrease the geometry detail based on camera distance.
  • Texture blending to smooth the transition between textures, to make it look more natural.
  • Applied render distance fog to the scene.
  • A report explaining how the techniques work and explaining what they are for was also created for this project.
